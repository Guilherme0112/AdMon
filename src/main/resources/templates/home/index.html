Perguntas frequentes

Como posso traçar metas usando o AdMon?

Nosso site oferece ferramentas profissionais para que você possa se planejar de maneira intuitiva e fácil. Você pode organizar tudo em um só lugar com o cadastro de ganhos e de contas, onde todo mês você tem os dados em um dashboard fácil de ler e entender. Dívidas ou saldos são repassados para o próximo mês caso você não consiga pagar.

Quando custa para usar o AdMon?

Atualmente você não precisa pagar nada para usar o nosso site, ele é totalmente gratuito para que você possa se planejar sem precisar gastar nada.

É seguro usar o AdMon?

Sim, nosso site criptografa os dados sensíveis do usuário. Também temos verificação por e-mail e por senha, garantindo que somente você possa alterar dados sensíveis da sua conta.

Como posso ajudar com sugestões para o site?

Você pode enviar um e-mail para teste@email.com e sugerir melhorias! Estamos sempre tentando oferecer as melhores ferramentas para você.
